// Introduction

Target acquired. This manual is your briefing for the upcoming Cyber Dojo CTF assessment. The mission details are classified, but this guide provides the necessary intelligence to prepare your arsenal.

// Prerequisites & Knowledge Base

1. Web Application Security

  • OWASP Top 10: Master the common vulnerabilities.
  • Injection Flaws: SQLi - understand how to manipulate queries.
  • XSS: Stored vs. Reflected. Know how to execute scripts in a victim's browser.
  • Broken Access Control & CSRF: Session hijacking and request forgery.

2. Network Security

  • Protocols: TCP/IP, HTTP, FTP.
  • Packet Analysis: Reading raw traffic (PCAP). Identifying cleartext credentials.

3. System Security & Cryptography

  • Memory Management: Stack, heap, and buffer overflows (C language).
  • Cryptography: RSA (primes, modulus, exponents). Weak key implementation.
  • Password Security: Hashing (MD5, SHA) and cracking techniques (john, hashcat).

4. Scripting & Automation

  • Python: Automating attacks and simulating malware.
  • Linux CLI: Navigation, compilation, and file management.

// Required Tools

  • Python 3
  • Wireshark or tcpdump
  • GCC (Compiler)
  • Text Editor/IDE (VS Code, Vim)
  • Browser with DevTools

// General Advice

  • Think like an attacker. What does the system not prevent?
  • Sanitize inputs? Try to break them.
  • Read error messages. They leak intel.
  • RTFM.
